# **Conditional and Looping Statements**

This topic groups conditional and looping constructs in Java. Subtopics in this folder are:

- **Conditional Statements** — `if`, `if-else`, `if-else if` ladder, and `switch`/`case`
- **Looping Statements** — `for`, enhanced `for` (for-each), `while`, and `do-while`

Examples include: simple calculators, odd/even checks, multiplication tables, pattern printing, and nested-loop examples. See the `Looping Statements/` and `Conditional Statements/` folders for runnable `.java` files.
