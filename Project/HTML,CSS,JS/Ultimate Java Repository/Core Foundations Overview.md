# 🛠️ Core Foundations

This section covers fundamental Java language concepts and the small, standalone programs located at the repository root:

- `Access_Modifiers.java`
- `Auto_Boxing_and_Auto_Unboxing.java`
- `Type_Casting.java`
- `Variables.java`
- `Wrapper_Class.java`

Core subtopics:

- **Variables** — declaration, primitive types, and scope
- **Type Casting** — implicit and explicit conversions between primitives and reference types
- **Access Modifiers** — `public`, `private`, `protected`, and package-private visibility
- **Wrapper Classes** — `Integer`, `Double`, `Character`, etc., and their usage
- **Auto-Boxing and Auto-Unboxing** — automatic conversion between primitives and wrapper objects

Each subtopic contains small example programs under the repository root or matching subfolders. Open the corresponding `.java` files to view runnable examples.
