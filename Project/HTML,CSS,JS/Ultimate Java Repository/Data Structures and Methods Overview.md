# 🧩 Data Structures

This topic covers common data structures and related language features in Java. The repository organizes Java examples in matching folders such as `Arrays/`, `Strings/`, and `Methods/`.

Core subtopics:

- **Arrays** — single- and multi-dimensional arrays, traversal, common algorithms (sorting/searching) implemented in Java
- **Strings** — `String`, `StringBuilder`, and `StringBuffer` usage, common operations and immutability
- **Methods** — method declaration, overloading, passing arrays and objects, return values, and recursion

Each subtopic contains small, focused `.java` examples in the corresponding folder. See `Arrays/`, `Strings/`, and `Methods/` for runnable programs and sample usage.
