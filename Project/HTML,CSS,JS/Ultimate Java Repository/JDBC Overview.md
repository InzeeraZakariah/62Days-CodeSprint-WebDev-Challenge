# **Java Database Connectivity (JDBC)**

This section contains JDBC example programs demonstrating how to connect Java applications to relational databases using JDBC drivers.

Typical examples in this repository:

- `Connect_to_MySQL_Database.java` — basic connection and query execution
- `CRUD_Operations.java` — Create, Read, Update, Delete examples using JDBC
- `Insert_Data_Using_INSERT_Query.java`, `Display_Data_Using_SELECT_Query.java`, `Modify_Data_Using_UPDATE_Query.java`, `Delete_Record_Using_DELETE_Query.java`

Notes:
- Update JDBC connection strings and credentials before running examples.
- Ensure the appropriate JDBC driver (for example, MySQL Connector/J) is on the classpath when compiling and running the examples.
