# **Object-Oriented Programming (OOP)**

This section groups the Object-Oriented Programming concepts and contains example programs and small demos illustrating each concept.

Core subtopics and sample locations in this repository:

- **Classes** — define and use classes (see `OOP/Classes/`)
- **Objects** — instantiate and interact with objects (see `OOP/Objects/`)
- **Constructors** — default, parameterized, and copy constructors (see `OOP/Constructors/`)
- **Inheritance** — single, multilevel, and hierarchical examples (see `OOP/Inheritance/`)
- **Interfaces** — defining and implementing interfaces
- **Polymorphism** — compile-time (overloading) and runtime (overriding) examples
- **Method Overloading and Overriding** — various overload/override examples
- **Exception Handling** — `try`/`catch`/`finally`, custom exceptions (see `Exception Handling/`)
- **Threads and Runnable** — `Thread` and `Runnable` examples (see `Threads and Runnable/`)

Open the relevant subfolders to view focused `.java` examples and run them with `javac` and `java`.
