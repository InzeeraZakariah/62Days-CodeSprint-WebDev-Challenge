# 🔩 Bash Fundamentals Overview

This section explores the fundamental concepts of Bash (Bourne Again SHell) scripting. It covers everything from basic syntax and commands to core concepts like variables and file permissions.

**Bash** is a powerful command-line interpreter and scripting language that is the default shell on most Linux distributions and macOS. It's an essential tool for developers, system administrators, and power users for automating tasks.

---

### Core Subtopics:

- **Basic Syntax**: Learn the basic structure of a shell script, including shebangs, comments, and executing commands.
- **Variables**: Discover how to declare, assign, and use variables to store data within your scripts.
- **Permissions**: Understand the Linux file permission model and how to use `chmod` to make your scripts executable.

Each subtopic contains code examples within its respective folder.
